# OSINT-tool
okay so i made this thing called RECON and it's basically a domain/IP lookup tool but make it aesthetic. type in any website, it pulls the DNS records, SSL certs, GeoIP location, HTTP headers, threat intel — the whole situationship. dark terminal UI, animated scan logs, it literally feels like you're in a movie fr

Features

DNS Enumeration — Live A, MX, NS, and TXT record lookups via Cloudflare DNS-over-HTTPS
WHOIS / RDAP — Domain registration info including registrar, org, creation and expiry dates
GeoIP Intelligence — IP geolocation with city, country, ASN, and timezone
SSL / TLS Analysis — Certificate issuer, subject, expiry, protocol version, and cipher suite
HTTP Headers Inspection — Server, security headers, and powered-by detection
Tech Stack Detection — Fingerprinting common frameworks and infrastructure
Threat Intel Score — Aggregated risk scoring across malware, phishing, and spam signals
Subdomain Discovery — Common subdomain enumeration with active/inactive status
Scan Modes — Full scan or focused modules: DNS, SSL, Headers, GeoIP
Export — Copy raw JSON or download a self-contained HTML report
